CSS Techniques Demonstration

Cette page démontre toutes les techniques CSS apprises dans le cours. Chaque section illustre un concept différent avec des exemples concrets et visuels.

1. Inline, Internal et External Styles

External Stylesheet (Fichier main.css)

Ce paragraphe utilise les styles du fichier externe css_style/main.css. C'est la méthode recommandée pour organiser vos styles CSS.

Internal Styles (Balise <style> dans le <head>)

Ce bloc utilise des styles internes définis dans la balise <style> en haut de cette page. Utile pour des styles spécifiques à une seule page.

Inline Styles (Attribut style)

Ce paragraphe utilise un style inline directement dans l'attribut style de la balise. À éviter en général, mais utile pour des modifications rapides ou dynamiques.

5. Relationship Selectors

Descendant Selector (nav a)

Dans la navigation en haut, tous les liens à l'intérieur de <nav> sont stylés avec le sélecteur "nav a" (descendant).

Child Selector (nav > ul > li)

Les éléments <li> qui sont des enfants directs de <ul> dans <nav> utilisent "nav > ul > li" (child selector).

Adjacent Sibling Selector (h2 + p)

11. Floating Elements et Clearing

Float Left

Cette image flotte à gauche du texte. Le texte s'enroule naturellement autour de l'image grâce à la propriété float: left. C'est très utile pour créer des layouts de magazine ou des articles avec des images intégrées dans le texte.

Float Right

Cette image flotte à droite du texte avec float: right. Le texte s'enroule maintenant du côté gauche de l'image, créant un effet visuel différent tout en restant harmonieux.

La propriété clear est utilisée pour empêcher que les éléments suivants ne s'enroulent autour des éléments flottants. Le clearfix est une technique pour contenir les floats.

12. Specificity Rule (Règle de spécificité)

Hiérarchie de spécificité

Paragraphe normal (element selector - faible spécificité)

Paragraphe avec classe (class selector - moyenne spécificité)

Paragraphe avec ID (ID selector - haute spécificité)

Paragraphe avec !important (priorité maximale - à éviter!)

Ordre de spécificité (du plus faible au plus fort):
  1. Element selectors (p, h1, div)
  2. Class selectors (.class-name)
  3. ID selectors (#id-name)
  4. Inline styles (style="...")
  5. !important (priorité absolue)

14. Inline vs Block vs Inline-Block Elements

Display: inline

Les éléments inline ne cassent pas le flux du texte et s'affichent sur la même ligne.

Display: block

Premier élément block - prend toute la largeur
Deuxième élément block - commence une nouvelle ligne

Display: inline-block

Inline-block 1 Inline-block 2 Inline-block 3

Les éléments inline-block se comportent comme inline mais acceptent width/height.
